# Web Design Optimization Skill

网页设计与优化技能，用于创建和优化各类网页界面，包括登录界面设计、动态背景管理、按钮 UI 优化、响应式布局等功能。

## ✨ 功能特点

- 🎨 登录界面设计 - 左侧登录区域，动态背景，视差效果
- 🖼️ 动态背景管理 - 支持图片和视频背景，轮播功能
- 🔘 界面元素优化 - 现代按钮设计，响应式布局
- 🖱️ 交互体验优化 - 触摸支持，全屏模式
- ⚙️ 后台运行支持 - 服务端部署，进程管理

## 📦 安装

```bash
git clone https://github.com/2023winner/web-design-optimization-skill.git
cd web-design-optimization-skill
pip install flask flask-socketio
```

## 🚀 使用方法

### 启动服务

```bash
python app.py
```

然后在浏览器打开 `http://localhost:5000`

### 自定义背景

在设置页面上传背景图片或视频即可。
